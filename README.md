# 📊 Vrinda Store Annual Report 2022

## 📖 Project Description
This project is an interactive data analysis dashboard created using Excel. It provides insights into the sales performance of Vrinda Store for the year 2022. The dashboard helps in understanding customer behavior, sales trends, and order status.

---

## 🎯 Key Features
- Monthly **Order vs Sales analysis**
- Gender-based sales comparison (Men vs Women)
- Order status breakdown (Delivered, Cancelled, Returned, Refunded)
- Top 5 performing states by sales
- Age group vs gender analysis
- Sales distribution across different channels
- Interactive filters for:
  - Month
  - Category (Kurta, Saree, Set, Top)
  - Channel (Flipkart, Amazon, Meesho, etc.)

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar, Pie, Line)
- Slicers for interactivity

---

## 📊 Insights from Dashboard
- Women contribute **64%** of total sales, higher than men.
- Maximum orders are **successfully delivered (92%)**.
- **Maharashtra** is the top-performing state (~2.99M sales).
- **Amazon** is the leading sales channel (~35%).
- Adult women contribute the highest share of orders.

---

## 🚀 How to Use
1. Download the Excel file
2. Open in Microsoft Excel
3. Use slicers (Month, Category, Channel) to filter data
4. Analyze charts for insights
